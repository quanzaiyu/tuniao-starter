# Quanzaiyu's Uniapp Vite4 Vue3 Scaffold

## 运行环境

- node: >=16.0.0
- pnpm: >=7.0.0

## 安装依赖

安装依赖：

```
pnpm i
```

## 运行项目

### 运行到H5

```
pnpm dev
```

运行地址：http://localhost:8000/


或者使用：

```
pnpm dev:h5
```

运行地址：http://localhost:5173/


### 运行到支付宝小程序

```
pnpm dev:mp-alipay
```

构建路径：

```
项目路径\dist\dev\mp-alipay
```

## 查看文档

```
pnpm docs:dev
```

文档地址：http://localhost:8081/
